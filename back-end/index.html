Following are the api which may be help full to get started.

  1. To set cookies use following api.

    http://localhost:5000/cookie/setcookie/:key/:value(use get method)

  2. To read cookies use following api.

    http://localhost:5000/cookie/readcookies

  3. To add a product into database use following api.

    http://localhost:5000/product/postproduct

  4. To get all products from database use following api.

    http://localhost:5000/product

  5. To edit a product use following api.

    http://localhost:5000/product/:id (use put method)

  6. To delete a product use following api.

    http://localhost:5000/product/:id (use delete method)

  7. To get a product specified by user use following api.

    http://localhost:5000/product/userproducts/:id

  8. To register a new user use following api.

    http://localhost:5000/registration (use post method)

  9. To get all user use following api.

    http://localhost:5000/registration (use get method)

  10. To do login use following api.

    http://localhost:5000/registration/login (use post method)
